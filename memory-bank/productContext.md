# Product Context

## Зачем
Быстро находить потенциальные разворотные/продолжительные точки по структуре рынка (SMC) на заданном таймфрейме и получать уведомления в Telegram.

## Как должно работать
- Берём OHLCV по символам (фильтруем по ликвидности/объёму).
- Считаем структуру (pivot → CHOCH/BOS).
- Если на последней закрытой свече есть подтверждённый BOS после CHOCH — отправляем картинку + подпись.
