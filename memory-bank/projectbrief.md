# Project Brief

## Цель
Скрипт-сканер (Bybit) для поиска SMC-сигнала **CHOCH → BOS** на выбранном таймфрейме (`TIMEFRAME`) и отправки алертов/графиков в Telegram.

## Ключевые требования
- Источник свечей: Bybit v5 `market/kline` (spot по умолчанию).
- Таймфрейм задаётся через `.env` (`TIMEFRAME=D|W|...`) и должен влиять на **все** расчёты.
- Сигнал: BOS после CHOCH, причём BOS должен случиться на **последней закрытой** свече.
