# Active Context

## Текущий фокус
- Убедиться, что `TIMEFRAME` влияет на расчёты CHOCH/BOS и вспомогательные фильтры (например, объём).

## Недавние изменения
- Нормализация `TIMEFRAME` (поддержка `D/W/M` и `1D/1W/1M`).
- Фильтр по объёму теперь считает средний объём **за N баров** текущего таймфрейма и корректно пишет единицы (дни/недели/месяцы/бары).
